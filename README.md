# VUT FEKT ALD zaznamy z kovidu
Web pro lepší sledování záznamů přednášek a cvičení BPC-ALD (předmět VUT FEKT) z kovidu.
## Podrobněji:
Tento projekt je jednoduchá html stránka, která má sloužit pro lepší přístup k přednáškám a cvičením z BPC-ALD, které byly nahráy za kovidu. Není to oficiální zdroj, pouze studentský projekt.

PS: Pokud máte nějaký nápad na zlepšení, neváhejte se mi ozvat.
